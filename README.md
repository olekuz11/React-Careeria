# Northwind React - Frontend Tests

## Running the tests

Install dependencies:
```
npm install
```

Run tests:
```
npm test
```

## Test setup notes
- Jest 27.5.1
- jest-environment-jsdom@27 
- @testing-library/react
- @testing-library/jest-dom

## Tests included
- Message.test.jsx - Message component rendering
- Customer.test.jsx - Customer component rendering and button visibility
- CustomerAdd.test.jsx - Customer add form rendering and cancel button
- ProductAdd.test.jsx - Product add form rendering and cancel button
- EmployeeAdd.test.jsx - Employee add form rendering and cancel button
